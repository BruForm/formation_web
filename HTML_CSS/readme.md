## FAQ : 
J'ai fait 2-3 tests pour confirmer certaines choses que Baptiste m'avait demandé hier pendant le cours. Donc pour éviter les confusions, voici le résultat :

-"place-content" est une propriété faites pour Grid qui est un raccourci pour "justify-content + align-content". Baptiste me demandait si l'on pouvait s'en servir avec Flex. J'ai testé : NON. Ca ne remplace pas le justify-content + align-items de Flex !!! Donc continuez d'utiliser Flex comme vous l'avez appris jusqu'ici.

-créer des fonctions en CSS : je trouve rien sur le sujet donc c'est encore certainement TRES expérimental. Ne vous en souciez pas (par contre vous pouvez déjà le faire en SCSS)

-Baptiste disait qu'il n'avait pas eu besoin d'utiliser de  "position" pour pouvoir donner un "z-index" à un élément. J'ai essayé de reproduire la situation et.... c'est FAUX. Assigner une position autre que "static" est nécessaire. La raison pour laquelle ca a marché avec Baptiste est probablement parce qu'il manipulait des éléments Flex ( ou Grid ) qui sont des cas particuliers (voir le screenshot ci dessous)

-@import en CSS (sans SCSS) ---> oui, c'est possible d'importer du CSS dans du CSS (de la même façon que l'on importe des polices d'écriture par exemple) mais c'est beaucoup moins optimisé que d'avoir du SCSS qui compile tous vos imports en un seul fichier CSS. Donc le SCSS reste meilleur ^^
