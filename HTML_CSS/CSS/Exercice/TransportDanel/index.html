    a propos

    Créée en 1969, TRANSPORTS DANEL est une entreprise familiale spécialisée dans le transport de marchandises sous température dirigée et autre types de transports.

    Pour notre équipe de 45 collaborateurs, il est important d'assurer un service de qualité qui reponde à vos besoins.

    Installés dans le départements du Puy-de-Dome, nous effectuons également des transports de marchandises partout en France.
